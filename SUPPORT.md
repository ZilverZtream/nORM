# Support Policy

## Current Status

nORM is pre-v1. The `main` branch and latest preview package are the supported
development line.

## v1 Versioning

After `1.0.0`:

- patch releases contain compatible bug, security, and documentation fixes,
- minor releases may add compatible APIs or provider capabilities,
- major releases may contain breaking API or behavior changes,
- deprecated APIs will include documented replacement guidance before removal.

## Getting Help

Use GitHub issues for reproducible bugs and GitHub discussions for usage
questions. Include provider, driver version, .NET SDK/runtime version, schema,
query or migration snippet, and the smallest reproduction possible.

Security issues must follow `SECURITY.md` instead of public issues.
