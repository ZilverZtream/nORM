## Problem

NH-1001 proved scaffolded output COMPILES for every supported shape (798 in-proc Roslyn checks)
and pinned the options/safety surface. What no suite pins: SEMANTIC fidelity - that the
scaffolded model actually DESCRIBES the schema it was scaffolded from. A scaffolder can emit
compiling code that silently drops nullability, mistypes a column, or loses a primary key; only
a round-trip oracle catches that class. This matrix closes the fidelity cells: scaffold a
database, compile the output in-proc, rebuild a schema from the scaffolded model, re-create it,
and compare the two live schemas structurally.
