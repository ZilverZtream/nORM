# Changelog

All notable changes to nORM are tracked here. Release entries should follow
semantic versioning and separate breaking changes, features, fixes, performance,
security, and documentation.

## Unreleased

### Breaking / API

- Public API is under v1 review and may still change before `1.0.0`.

### Features

- Added v1 contracts for provider capabilities, temporal versioning, bulk
  operations, optimistic concurrency, source generation, and production
  operations.

### Performance

- Added benchmark governance and fair Raw ADO.NET baseline categories for
  convenience, optimized, and prepared-optimized manual readers.

### Security

- Documented multi-tenancy, logging redaction, raw SQL, and vulnerability
  reporting boundaries.

### Documentation

- Added v1 operational and release-governance docs.
